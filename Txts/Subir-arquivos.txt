## Primeira vez (Configuração)
git init
git remote add origin <URL>
git config --global user.name ""
git config --global user.email ""

## Alterar a branch
git branch -M nome

## Atualizar
git init
git add .
git commit -m ""
git push -u origin master

## Clonar
(no cmd)
git clone <URL>

## Sincronizar
git pull
